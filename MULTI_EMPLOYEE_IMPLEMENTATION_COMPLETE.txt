STATUS-BASED GROUPING

  In Progress Section:
    Conditions: watchProgress > 0 AND watchProgress < 100
    Example: Data Visualization at 92% (Casey)
    Color: Yellow divider
    Badge: ⟳ In Progress

  To Start Section:
    Conditions: watchProgress = 0
    Example: Python Basics (Casey), Data Viz (Sam)
    Color: Blue divider
    Badge: ⊕ To Start

  Completed Section:
    Conditions: watchProgress = 100
    Example: Python Basics 100% (Morgan)
    Color: Green badge
    Badge: ✓ Completed

API METHODS - NEW

  PrototypeAPI.getEmployees()
    Returns: Array of all employees
    Example: [{ id, firstName, lastName, role, email }, ...]

  PrototypeAPI.getEmployeeVideos(employeeId)
    Returns: Videos for specific employee
    Example: Calling with 'emp-sam' returns Sam's 3 videos

  PrototypeAPI.setSelectedEmployee(employeeId)
    Purpose: Store selected employee in sessionStorage
    Effect: Persists across page refreshes

  PrototypeAPI.getSelectedEmployeeId()
    Returns: Currently selected employee ID
    Default: 'emp-casey'

  PrototypeAPI.getSelectedEmployee()
    Returns: Full employee object (id, name, role, email)
    Usage: Display employee details
