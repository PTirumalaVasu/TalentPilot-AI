### JavaScript

```javascript
async function loadDashboard() {
  const assignments = await PrototypeAPI.getAssignments();
  document.getElementById('skills-dashboard-summary-count').textContent =
    `Total: ${assignments.length} assignment${assignments.length === 1 ? '' : 's'}`;

  const tbody = document.getElementById('assignments-tbody');
  tbody.innerHTML = assignments.map(rowHtml).join('');
}

function provenanceBadge(a) {
  const styles = {
    'Verified': 'bg-green-100 text-green-800',
    'Self-reported': 'bg-gray-100 text-gray-700',
    'Needs Attention': 'bg-amber-100 text-amber-800',
  };
  const label = `${a.provenance} · ${a.lastUpdate}`;
  return `<span class="inline-flex items-center px-2 py-0.5 rounded-full text-xs font-medium ${styles[a.provenance]}">${escapeHtml(label)}</span>`;
}

function progressBar(percent) {
  return `
    <div class="w-24 h-2 bg-gray-100 rounded-full overflow-hidden">
      <div class="h-full bg-talentpilot-600" style="width: ${percent}%"></div>
    </div>`;
}
```

### Tailwind Classes to Use

- **Table**: `w-full`, `border-collapse`, `bg-white`, `rounded-lg`, `shadow-sm`
- **Provenance badges**: `bg-green-100 text-green-800` (Verified) · `bg-gray-100 text-gray-700` (Self-reported) · `bg-amber-100 text-amber-800` (Needs Attention) — always paired with text, never color-only
- **Progress bar**: `bg-gray-100` track, `bg-talentpilot-600` fill

## Common Issues & Fixes

### Issue: Summary count shows "Total: undefined assignments"
**Symptom**: `assignments.length` is undefined
**Cause**: `PrototypeAPI.getAssignments()` didn't resolve before render
**Fix**: Ensure `loadDashboard()` is `async` and awaited from `initPage()`
