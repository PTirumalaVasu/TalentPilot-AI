### JavaScript

```javascript
async function onSkillSelected() {
  const val = document.getElementById('skill-assignment-flow-skill-search').value;
  assignmentFlowState.skillId = val || null;
  document.getElementById('skill-assignment-flow-btn-continue-step2').disabled = !val;

  if (val) {
    assignmentFlowState.content = await PrototypeAPI.getContentForSkill(val);
  }
}
```

**Update `openAssignmentFlow()`** to also populate the skill dropdown:

```javascript
const skills = await PrototypeAPI.getSkills();
const skillDropdown = document.getElementById('skill-assignment-flow-skill-search');
skillDropdown.innerHTML = '<option value="">Search for a skill or select...</option>' +
  skills.map(s => `<option value="${s.id}">${escapeHtml(s.name)}</option>`).join('');
```

**Update `goToStep(n)`** to show/hide steps 1 and 2:

```javascript
function goToStep(n) {
  document.getElementById('skill-assignment-flow-progress-indicator').textContent = `Step ${n} of 3`;
  document.getElementById('asf-step-1').classList.toggle('hidden', n !== 1);
  document.getElementById('asf-step-2').classList.toggle('hidden', n !== 2);
}
```
