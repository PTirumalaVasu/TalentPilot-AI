## What to Build

### JavaScript only

```javascript
document.addEventListener('click', (e) => {
  const userMenu = document.getElementById('continue-watching-user-menu');
  const dropdown = userMenu.querySelector('[data-dropdown]');
  if (!userMenu.contains(e.target) && !dropdown.classList.contains('hidden')) {
    dropdown.classList.add('hidden');
  }
});
```
